Android学习必须自己实现的Demo之ContentProvider

发表于 2017-02-12 |

什么是内容提供者

  1. ContentProvider又名内容提供者，是Android四大组件之一，为存储和获取数据提供统一的接口。
  2. 可以在不同程序之间共享数据，Android中原本就存在许多以ContentProvider形式存储的数据，比如：联系人、照片、音频、视频等等。
  3. ContentProvider以表的形似存储数据，类似数据库的表。

为什么使用ContentProvider

  1. 可以共享数据，统一了数据的访问方式。
  2. 简化了与用户之间的数据库内容交换，数据变化时通知UI
  3. 可以更轻松的切换数据源，有大量的API用于优化同步和查询数据

学习使用ContentProvider的姿势

使用Content可以分为四个步骤

  1. 确定应用程序需要支持的Uri
  2. 创建并更新Contract协议类，将Uri以及一些表的信息包括在内。
  3. 填写支持各个Uri的UriMatcher
  4. 实现ContentProvide的六个必须函数
    • onCreate() 用来初始化ContentProvider
    • getType(Uri) 返回内容提供程序中的MIME类型的数据
    • insert(Uri, ContentValues) 插入新数据到ContentProvider
    • delete(Uri, String, String[]) 从ContentProvider中删除数据
    • update(Uri, ContentValues, String, String[]) 更新ContentProvider中的数据
    • query(Uri, String[], String, String[], String) 查询ContentProvider中的数据

什么是Uri

Uri是统一资源标识符的意思，ContentProvider允许我们根据Uri来考虑跟视图相关的数据，简单的说就是：Uri可以标识资源，资源可以是数据库中的一行或者多行数据。
例如：content://com.example.t.mysamplereels/movie/

  • content: 表示的是引用ContentProvider
  • 内容授权： com.example.t.mysamplereels，用于表示内容提供者的唯一字符串，它其实是应用程序的包名。
  • movie: 数据库中的movie表

创建Contract协议类

Contract是数据模型、存储和视图间的一种协议，说明了如何访问信息。下面是自己编写的一个例子

public static final class MovieReviewEntry implements BaseColumns {
  //content://com.example.t.mysamplereels/movie_review
  public static final Uri CONTENT_URI = BASE_CONTENT_URI.buildUpon().appendPath
      (PATH_MOVIE_REVIEW).build();

  public static final String CONTENT_TYPE = ContentResolver.CURSOR_DIR_BASE_TYPE + "/"
      + CONTENT_AUTHORITY + "/" + PATH_MOVIE_REVIEW;

  public static final String CONTENT_ITEM_TYPE = ContentResolver.CURSOR_ITEM_BASE_TYPE + "/"
      + CONTENT_AUTHORITY + "/" + PATH_MOVIE_REVIEW;

  public static final String TABLE_NAME = "review";
  //外键
  public static final String COLUMN_MOVIE_ID = "movie_id";
  //评论的id
  public static final String COLUMN_REVIEW_ID = "review_id";
  //评论作者
  public static final String COLUMN_REVIEW_AUTHOR = "review_author";
  //评论内容
  public static final String COLUMN_REVIEW_CONTENT = "review_content";
  //评论链接
  public static final String COLUMN_REVIEW_URL = "review_url";

  //insert的时候调用
  public static Uri buildMovieReviewUri(long id) {
    return ContentUris.withAppendedId(CONTENT_URI, id);
  }

  //content://com.example.t.mysamplereels/movie_review/movieId
  public static Uri buildMovieIdForReview(String movieId) {
    return CONTENT_URI.buildUpon().appendPath(movieId).build();
  }

  //获取movie_id字段
  public static String getMovieIdFromUri(Uri uri) {
    return uri.getPathSegments().get(1);
  }
}

填写UriMatcher

UriMatcher主要用于添加Uri,并匹配Uri。在ContentProvider中将要使用的Uri全部添加上。

final UriMatcher uriMatcher = new UriMatcher(UriMatcher.NO_MATCH);
//content://com.example.t.mysamplereels
final String authority = MovieContract.CONTENT_AUTHORITY;

// movie表的所有记录
//content://com.example.t.mysamplereels/movie
uriMatcher.addURI(authority, MovieContract.PATH_MOVIE, MOVIE);
//movie表的下的所有记录
//content://com.example.t.mysamplereels/movie/*---->MOVIE
//uriMatcher.addURI(authority, MovieContract.PATH_MOVIE + "/*", MOVIE);
//movie表的某个记录
//content://com.example.t.mysamplereels/movie/*---->MOVIE_WITH_ID
uriMatcher.addURI(authority, MovieContract.PATH_MOVIE + "/*", MOVIE_WITH_ID);

//movie_video表的所有记录
//content://com.example.t.mysamplereels/movie_video---->MOVIE_VIDEO
uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_VIDEO, MOVIE_VIDEO);
uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_VIDEO + "/*", MOVIE_VIDEO_WITH_ID);
//movie_review表的所有记录
//content://com.example.t.mysamplereels/movie_review---->MOVIE_VIDEO
uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_REVIEW, MOVIE_REVIEW);
uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_REVIEW + "/*", MOVIE_REVIEW_WITH_ID);

添加完需要匹配的Uri后，就可以使用uriMatcher.match(uri)方法对输入的Uri进行匹配

实现getType(Uri uri)

通过getType来匹配不同的数据源

@Nullable @Override public String getType(Uri uri) {
    final int match = sUriMatcher.match(uri);
    switch (match) {
      case MOVIE:
        return MovieContract.MovieEntry.CONTENT_TYPE;
      case MOVIE_WITH_ID:
        return MovieContract.MovieEntry.CONTENT_ITEM_TYPE;
      case MOVIE_VIDEO:
        return MovieContract.MovieVideoEntry.CONTENT_TYPE;
      case MOVIE_VIDEO_WITH_ID:
        return MovieContract.MovieVideoEntry.CONTENT_TYPE;
      case MOVIE_REVIEW:
        return MovieContract.MovieReviewEntry.CONTENT_TYPE;
      case MOVIE_REVIEW_WITH_ID:
        return MovieContract.MovieReviewEntry.CONTENT_TYPE;
      default:
        throw new UnsupportedOperationException("Unknown uri: " + uri);
    }
  }

数据的插入

@Nullable @Override public Uri insert(Uri uri, ContentValues values) {
  SQLiteDatabase db = mDbHelper.getWritableDatabase();
  final int match = sUriMatcher.match(uri);
  Uri returnUri;
  long _id;
  switch (match) {
    case MOVIE:
      _id = db.insert(MovieContract.MovieEntry.TABLE_NAME, null, values);
      if (_id > 0) {
        returnUri = MovieContract.MovieEntry.buildMovieUri(_id);
      } else {
        throw new SQLException("Failed to insert row into  " + uri);
      }
      break;
    case MOVIE_VIDEO:
      _id = db.insert(MovieContract.MovieVideoEntry.TABLE_NAME, null, values);
      if (_id > 0) {
        returnUri = MovieContract.MovieVideoEntry.buildMovieVideoUri(_id);
      } else {
        throw new UnsupportedOperationException("Unknown uri: " + uri);
      }
      break;
    case MOVIE_REVIEW:
      _id = db.insert(MovieContract.MovieReviewEntry.TABLE_NAME, null, values);
      if (_id > 0) {
        returnUri = MovieContract.MovieReviewEntry.buildMovieReviewUri(_id);
      } else {
        throw new UnsupportedOperationException("Unknown uri: " + uri);
      }
      break;
    default:
      throw new UnsupportedOperationException("Unknown uri: " + uri);
  }
  getContext().getContentResolver().notifyChange(uri, null);
  return returnUri;
}

public Cursor query(Uri uri, String[] projection, String selection, String[] selectionArgs,
    String sortOrder) {
  SQLiteDatabase db = mDbHelper.getWritableDatabase();
  Cursor retCursor;
  String movieId;
  switch (sUriMatcher.match(uri)) {
    case MOVIE: {    //查询所有记录
      retCursor =
          db.query(MovieContract.MovieEntry.TABLE_NAME,
              projection,
              selection,
              selectionArgs,
              null,
              null,
              sortOrder);
      break;
    }
    case MOVIE_WITH_ID: {    //查询单个记录
      movieId = MovieContract.MovieEntry.getMovieId(uri);
      retCursor = db.query(MovieContract.MovieEntry.TABLE_NAME,
          projection,
          MovieContract.MovieEntry.COLUMN_MOVIE_ID + "=?",
          new String[] {movieId},
          null,
          null,
          sortOrder);
      break;
    }
    case MOVIE_VIDEO: {
      retCursor = db.query(MovieContract.MovieVideoEntry.TABLE_NAME,
          projection, selection, selectionArgs, null, null, sortOrder);
      break;
    }
    case MOVIE_VIDEO_WITH_ID: {//查询movie_id在预告片表的所有记录
      movieId = MovieContract.MovieVideoEntry.getMovieIdFromUri(uri);
      retCursor = sVideosByMovieIdQueryBuild.query(mDbHelper.getReadableDatabase(),
          projection,
          sMovieSelectionOfVideos,
          new String[] {movieId},
          null,
          null,
          sortOrder);
      break;
    }
    case MOVIE_REVIEW: {
      retCursor = db.query(MovieContract.MovieReviewEntry.TABLE_NAME,
          projection, selection, selectionArgs, null, null, sortOrder);
      break;
    }
    case MOVIE_REVIEW_WITH_ID: {  //查询movie_id在评论表的所有记录
      movieId = MovieContract.MovieReviewEntry.getMovieIdFromUri(uri);
      retCursor = sReviewsByMovieIdQueryBuild.query(mDbHelper.getReadableDatabase(),
          projection,
          sMovieSelectionOfReviews,
          new String[] {movieId},
          null,
          null,
          sortOrder);
      break;
    }
    default:
      throw new UnsupportedOperationException("Unknown uri: " + uri);
  }
  retCursor.setNotificationUri(getContext().getContentResolver(), uri);
  return retCursor;
}

以上的代码就是实现一个基本的ContentProvider，注意，因为ContentProvider是Android中的四大组件之一，必须在AndroidManifest文件中进行注册。下面是ContentProvider的完整代码

public class MovieProvider extends ContentProvider {

  public static final UriMatcher sUriMatcher = buildUriMatcher();
  public static final String TAG = MovieProvider.class.getSimpleName();
  private MovieDbHelper mDbHelper;

  public static final int MOVIE = 100;
  public static final int MOVIE_WITH_ID = 101;

  public static final int MOVIE_VIDEO = 200;
  public static final int MOVIE_VIDEO_WITH_ID = 201;

  public static final int MOVIE_REVIEW = 300;
  public static final int MOVIE_REVIEW_WITH_ID = 301;

  private static final SQLiteQueryBuilder sVideosByMovieIdQueryBuild;
  private static final SQLiteQueryBuilder sReviewsByMovieIdQueryBuild;

  static {
    sVideosByMovieIdQueryBuild = new SQLiteQueryBuilder();
    sReviewsByMovieIdQueryBuild = new SQLiteQueryBuilder();

    //内联 movie表 、movie_video表
    //movie_video inner join movie on movie_video.movie_id = movie._id;
    sVideosByMovieIdQueryBuild.setTables(MovieContract.MovieVideoEntry.TABLE_NAME + " INNER JOIN "
        + MovieContract.MovieEntry.TABLE_NAME
        + " ON " + MovieContract.MovieVideoEntry.TABLE_NAME
        + "." + MovieContract.MovieVideoEntry.COLUMN_MOVIE_ID
        + " = " + MovieContract.MovieEntry.TABLE_NAME
        + "." + MovieContract.MovieEntry._ID);

    //内联 movie表 、 movie_review表
    //movie_review inner join movie on movie_review.movie_id = movie._id
    sReviewsByMovieIdQueryBuild.setTables(MovieContract.MovieReviewEntry.TABLE_NAME + " INNER JOIN "
        + MovieContract.MovieEntry.TABLE_NAME
        + " ON " + MovieContract.MovieReviewEntry.TABLE_NAME
        + "." + MovieContract.MovieReviewEntry.COLUMN_MOVIE_ID
        + " = " + MovieContract.MovieEntry.TABLE_NAME
        + "." + MovieContract.MovieEntry._ID);
  }

  //movie.movie_id = ?
  private static final String sMovieSelectionOfVideos = MovieContract.MovieEntry.TABLE_NAME
      + "."
      + MovieContract.MovieEntry.COLUMN_MOVIE_ID
      + " = ? ";

  //movie.movie_id = ?
  private static final String sMovieSelectionOfReviews = MovieContract.MovieEntry.TABLE_NAME
      + "."
      + MovieContract.MovieEntry.COLUMN_MOVIE_ID
      + " = ? ";

  public static UriMatcher buildUriMatcher() {
    final UriMatcher uriMatcher = new UriMatcher(UriMatcher.NO_MATCH);
    //content://com.example.t.mysamplereels
    final String authority = MovieContract.CONTENT_AUTHORITY;

    // movie表的所有记录
    //content://com.example.t.mysamplereels/movie
    uriMatcher.addURI(authority, MovieContract.PATH_MOVIE, MOVIE);
    //movie表的下的所有记录
    //content://com.example.t.mysamplereels/movie/*---->MOVIE
    //uriMatcher.addURI(authority, MovieContract.PATH_MOVIE + "/*", MOVIE);
    //movie表的某个记录
    //content://com.example.t.mysamplereels/movie/*---->MOVIE_WITH_ID
    uriMatcher.addURI(authority, MovieContract.PATH_MOVIE + "/*", MOVIE_WITH_ID);

    //movie_video表的所有记录
    //content://com.example.t.mysamplereels/movie_video---->MOVIE_VIDEO
    uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_VIDEO, MOVIE_VIDEO);
    uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_VIDEO + "/*", MOVIE_VIDEO_WITH_ID);
    //movie_review表的所有记录
    //content://com.example.t.mysamplereels/movie_review---->MOVIE_VIDEO
    uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_REVIEW, MOVIE_REVIEW);
    uriMatcher.addURI(authority, MovieContract.PATH_MOVIE_REVIEW + "/*", MOVIE_REVIEW_WITH_ID);
    return uriMatcher;
  }

  @Override public boolean onCreate() {
    mDbHelper = new MovieDbHelper(getContext());
    return true;
  }

  @Nullable @Override public String getType(Uri uri) {
    final int match = sUriMatcher.match(uri);
    switch (match) {
      case MOVIE:
        return MovieContract.MovieEntry.CONTENT_TYPE;
      case MOVIE_WITH_ID:
        return MovieContract.MovieEntry.CONTENT_ITEM_TYPE;
      case MOVIE_VIDEO:
        return MovieContract.MovieVideoEntry.CONTENT_TYPE;
      case MOVIE_VIDEO_WITH_ID:
        return MovieContract.MovieVideoEntry.CONTENT_TYPE;
      case MOVIE_REVIEW:
        return MovieContract.MovieReviewEntry.CONTENT_TYPE;
      case MOVIE_REVIEW_WITH_ID:
        return MovieContract.MovieReviewEntry.CONTENT_TYPE;
      default:
        throw new UnsupportedOperationException("Unknown uri: " + uri);
    }
  }

  @Nullable @Override public Uri insert(Uri uri, ContentValues values) {
    SQLiteDatabase db = mDbHelper.getWritableDatabase();
    final int match = sUriMatcher.match(uri);
    Uri returnUri;
    long _id;
    switch (match) {
      case MOVIE:
        _id = db.insert(MovieContract.MovieEntry.TABLE_NAME, null, values);
        if (_id > 0) {
          returnUri = MovieContract.MovieEntry.buildMovieUri(_id);
        } else {
          throw new SQLException("Failed to insert row into  " + uri);
        }
        break;
      case MOVIE_VIDEO:
        _id = db.insert(MovieContract.MovieVideoEntry.TABLE_NAME, null, values);
        if (_id > 0) {
          returnUri = MovieContract.MovieVideoEntry.buildMovieVideoUri(_id);
        } else {
          throw new UnsupportedOperationException("Unknown uri: " + uri);
        }
        break;
      case MOVIE_REVIEW:
        _id = db.insert(MovieContract.MovieReviewEntry.TABLE_NAME, null, values);
        if (_id > 0) {
          returnUri = MovieContract.MovieReviewEntry.buildMovieReviewUri(_id);
        } else {
          throw new UnsupportedOperationException("Unknown uri: " + uri);
        }
        break;
      default:
        throw new UnsupportedOperationException("Unknown uri: " + uri);
    }
    getContext().getContentResolver().notifyChange(uri, null);
    return returnUri;
  }

  @Override public int delete(Uri uri, String selection, String[] selectionArgs) {
    SQLiteDatabase db = mDbHelper.getWritableDatabase();
    final int match = sUriMatcher.match(uri);
    int rowDeleted;
    if (null == selection) selection = "1";
    switch (match) {
      case MOVIE:
        rowDeleted =
            db.delete(MovieContract.MovieEntry.TABLE_NAME, selection, selectionArgs);
        break;
      case MOVIE_VIDEO:
        rowDeleted = db.delete(MovieContract.MovieVideoEntry.TABLE_NAME, selection, selectionArgs);
        break;
      case MOVIE_REVIEW:
        rowDeleted = db.delete(MovieContract.MovieReviewEntry.TABLE_NAME, selection, selectionArgs);
        break;
      default:
        throw new UnsupportedOperationException("Unknown uri: " + uri);
    }

    if (rowDeleted != 0) {
      getContext().getContentResolver().notifyChange(uri, null);
    }
    return rowDeleted;
  }

  @Override
  public int update(Uri uri, ContentValues values, String selection, String[] selectionArgs) {
    SQLiteDatabase db = mDbHelper.getWritableDatabase();
    final int match = sUriMatcher.match(uri);
    int rowUpdated;
    switch (match) {
      case MOVIE:
        rowUpdated =
            db.update(MovieContract.MovieEntry.TABLE_NAME, values, selection, selectionArgs);
        break;
      case MOVIE_VIDEO:
        rowUpdated = db.update(MovieContract.MovieVideoEntry.TABLE_NAME, values, selection,
            selectionArgs);
        break;
      case MOVIE_REVIEW:
        rowUpdated = db.update(MovieContract.MovieReviewEntry.TABLE_NAME, values, selection,
            selectionArgs);
        break;
      default:
        throw new UnsupportedOperationException("Unknown uri: " + uri);
    }
    if (rowUpdated != 0) {
      getContext().getContentResolver().notifyChange(uri, null);
    }
    return rowUpdated;
  }

  @Nullable @Override
  public Cursor query(Uri uri, String[] projection, String selection, String[] selectionArgs,
      String sortOrder) {
    SQLiteDatabase db = mDbHelper.getWritableDatabase();
    Cursor retCursor;
    String movieId;
    switch (sUriMatcher.match(uri)) {
      case MOVIE: {    //查询所有记录
        retCursor =
            db.query(MovieContract.MovieEntry.TABLE_NAME,
                projection,
                selection,
                selectionArgs,
                null,
                null,
                sortOrder);
        break;
      }
      case MOVIE_WITH_ID: {    //查询单个记录
        movieId = MovieContract.MovieEntry.getMovieId(uri);
        retCursor = db.query(MovieContract.MovieEntry.TABLE_NAME,
            projection,
            MovieContract.MovieEntry.COLUMN_MOVIE_ID + "=?",
            new String[] {movieId},
            null,
            null,
            sortOrder);
        break;
      }
      case MOVIE_VIDEO: {
        retCursor = db.query(MovieContract.MovieVideoEntry.TABLE_NAME,
            projection, selection, selectionArgs, null, null, sortOrder);
        break;
      }
      case MOVIE_VIDEO_WITH_ID: {//查询movie_id在预告片表的所有记录
        movieId = MovieContract.MovieVideoEntry.getMovieIdFromUri(uri);
        retCursor = sVideosByMovieIdQueryBuild.query(mDbHelper.getReadableDatabase(),
            projection,
            sMovieSelectionOfVideos,
            new String[] {movieId},
            null,
            null,
            sortOrder);
        break;
      }
      case MOVIE_REVIEW: {
        retCursor = db.query(MovieContract.MovieReviewEntry.TABLE_NAME,
            projection, selection, selectionArgs, null, null, sortOrder);
        break;
      }
      case MOVIE_REVIEW_WITH_ID: {  //查询movie_id在评论表的所有记录
        movieId = MovieContract.MovieReviewEntry.getMovieIdFromUri(uri);
        retCursor = sReviewsByMovieIdQueryBuild.query(mDbHelper.getReadableDatabase(),
            projection,
            sMovieSelectionOfReviews,
            new String[] {movieId},
            null,
            null,
            sortOrder);
        break;
      }
      default:
        throw new UnsupportedOperationException("Unknown uri: " + uri);
    }
    retCursor.setNotificationUri(getContext().getContentResolver(), uri);
    return retCursor;
  }

  /*管理事务*/
  @Override public int bulkInsert(Uri uri, ContentValues[] values) {
    final SQLiteDatabase db = mDbHelper.getWritableDatabase();
    final int match = sUriMatcher.match(uri);
    switch (match) {
      case MOVIE_VIDEO: {
        db.beginTransaction();
        int returnCount = 0;
        try {
          for (ContentValues value : values) {

            long _id = db.insert(MovieContract.MovieVideoEntry.TABLE_NAME, null, value);
            if (_id != -1) {
              returnCount++;
            }
          }
          db.setTransactionSuccessful();
        } finally {
          db.endTransaction();
        }
        getContext().getContentResolver().notifyChange(uri, null);
        return returnCount;
      }
      case MOVIE_REVIEW: {
        db.beginTransaction();
        int returnCount = 0;
        try {
          for (ContentValues value : values) {
            long _id = db.insert(MovieContract.MovieReviewEntry.TABLE_NAME, null, value);
            if (_id != -1) {
              returnCount++;
            }
          }
          db.setTransactionSuccessful();
        } finally {
          db.endTransaction();
        }
        getContext().getContentResolver().notifyChange(uri, null);
        return returnCount;
      }
      case MOVIE: {
        db.beginTransaction();
        int returnCount = 0;
        try {
          for (ContentValues value : values) {
            long _id = db.replace(MovieEntry.TABLE_NAME, null, value);
            if (_id != -1) {
              returnCount++;
            }
          }
          db.setTransactionSuccessful();
        } finally {
          db.endTransaction();
        }
        getContext().getContentResolver().notifyChange(uri, null);
        return returnCount;
      }
      default:
        return super.bulkInsert(uri, values);
    }
  }
}
